## 🎓 RECURSOS ADICIONALES

### Archivos de código relacionados:
```
/actions/
  ├── registry.ts          # Definición de todas las acciones
  ├── handlers.ts          # Lógica de handlers
  └── types.ts             # TypeScript types

/components/
  ├── product-detail/
  │   └── ReportSheet.tsx  # Modal reporte listing
  ├── groups/
  │   └── ReportGroupSheet.tsx  # Modal reporte grupo
  ├── group-detail/
  │   ├── GroupDetailPage.tsx   # Página principal
  │   ├── GroupHeader.tsx       # Header con menú
  │   └── GroupSheetsProvider.tsx  # Provider de sheets
  └── messages/
      └── ChatView.tsx     # Chat con moderación

/lib/
  ├── useGroupSheets.tsx   # Zustand store para grupos
  └── chatStore.ts         # Zustand store para chats
```
